  ### HOW INSTALL
   Clone or download repositorie
   ``` 
   git clone https://github.com/berlaseep/Crow-ler-crawler-deep-web-navigator.git
   ```
  
   You need install Postgresql and Tor Expert Bundle
  
  ### Install postgres
  ```
  https://www.enterprisedb.com/downloads/postgres-postgresql-downloads
  ```
  Execute postgresql-18.1-1-windows-x64.exe use password postgres and port 5432

 ### Install tor
  ```
  https://archive.torproject.org/tor-package-archive/torbrowser/13.0.9/tor-expert-bundle-windows-x86_64-13.0.9.tar.gz
  ```
  Execute tor.exe
